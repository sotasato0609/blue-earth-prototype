# BLUE EARTH リニューアル — クリッカブル・プロトタイプ

提案用に、実際にクリック／タップして回遊できる静的サイトのモックです。
**デザイン確認・導線検証が目的**です。

掲載情報（店舗名・住所・電話・営業時間・施設・サービス・お知らせ・企業情報）は
**公式サイト https://blueearth.co.jp/ の掲載内容（2026-06-23取得）を基に**作成しています。
- 店舗：総合スポーツクラブ BLUE EARTH 7店舗＋24時間ジム MY-BODY 14店舗＝計21店舗を収録
- 料金：会員種別・金額は公式でも店舗ごとの案内のため、数値は持たせず「店舗により異なる」と明示
  （唯一、甲府店キッズの公開料金のみ料金ページに実例として掲載）
- 写真：手元の汎用素材を割当（実店舗写真ではありません）
- キャンペーン等で公式に金額・実施の記載がないものは創作せず、表現を一般化しています

## 起動方法

ブラウザのセキュリティ制約を避けるため、簡易サーバ経由での閲覧を推奨します。

```bash
cd prototype
python3 -m http.server 8848
# → ブラウザで http://localhost:8848/ を開く
```

または同梱スクリプト:

```bash
./start.command   # macOS（ダブルクリックでも可）
```

> 各HTMLを直接ダブルクリック（file://）しても概ね動作しますが、一部ブラウザで
> 表示が崩れる場合があるため上記サーバ経由を推奨します。

## ページ構成（全14ページ・全リンク有効）

| ファイル | 内容 |

| `stores.html` | 店舗一覧（**エリア絞り込み**動作） |

| `join.html` | WEB入会（**3ステップフォーム**） |

| ファイル | 内容 |
|---|---|
| `mybody.html` | MY-BODYトップ |

## 動く要素（インタラクション）
- グローバルナビ ＋ サービス・ドロップダウン
- モバイル：ハンバーガー → ドロワーメニュー
- 店舗一覧：エリアタブで絞り込み／URL `?area=山梨県` で直リンク
- 店舗詳細：`?id=` で店舗データを切替
- 無料体験・WEB入会：フォーム送信モック（実送信なし）

## 構成
```
prototype/
├─ *.html              … 各ページ（<main>の中身のみ。ヘッダー/フッターはJSで共通注入）
├─ assets/css/be.css   … 共有スタイル（site-be / site-mb テーマ）
├─ assets/js/site.js   … ヘッダー/フッター注入＋全インタラクション
├─ assets/data/stores.js … 店舗・お知らせのサンプルデータ
└─ assets/img/         … 画像
```

## データの差し替え
店舗・お知らせは `assets/data/stores.js` の `window.STORES` / `window.NEWS` を編集すれば
店舗一覧・絞り込み・店舗詳細・トップのお知らせに一括反映されます（各 store に
`name/pref/area/addr/tel/hours/svc[]/mybody/img` を保持）。

## 方向転換について
ヘッダー/フッター/ナビは `assets/js/site.js`、配色は `assets/css/be.css`、
店舗・お知らせデータは `assets/data/stores.js` に集約しています。
方向性が変わっても、これらの共通ファイルの差し替えで全ページに一括反映できます。
